# Dependencies
Requires pytest

# How to use
run main.py
```python3 main.py```

Programs can be run via the following methods:
- Entering the file path to the program you would like to run and pressing the ">" button.
- Pressing the "Load" button and selecting a file from your system.
- Typing your own code into the Memory Panel.

When typing your own code, you can right click on your code to open a menu with the following actions:
- Copy
- Cut
- Paste

## RUN
When you are ready to run your program, press RUN to execute the program.
If the program requires user input, you will be prompted to give it.

## RESET
Press RESET to clear the currently-stored program from the app.
If you want to run a program again, you must reload it or re-enter your own code.

## SAVE
Press SAVE to save the current contents of your program (displayed in the Memory Panel)
to a text file.

## MISC
The app's color theme can be changed by accessing "Theme" from the top menu bar and
selecting your desired theme. A user guide can be found by accessing the "Help" button
on the menu bar.

This app supports files with either 4-digit words or 6-digit words.
Please note that files with a mix of both 4 and 6-digit words are not allowed.
Any errors will either appear as a popup notification or in the Console Panel.

## How to use unittests
run unittests.py
```pytest -v unittests.py```
