# Dependencies
No dependencies
# How to use
run test.py
```python3 test.py```
input your program, instruction by instruction as prompted in the console
enter -99999 to finish entering your program
your program will then run
if your program requires input (from READ instructions) you will be prompted to input a word
after finishing executing the UVSim will perform a memory dump, printing the contents of its registers and memory to the console
